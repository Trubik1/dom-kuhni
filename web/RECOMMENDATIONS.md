| Решение | Для чего | Почему |

## 5. A/B-тесты (что тестировать в первую очередь)

1. **Заголовок Hero:** «Создайте кухню, которая будет вдохновлять» vs «Кухни на заказ премиум-класса»
2. **Цвет CTA:** Синий (#2D5BFF) vs Золотой (#C8A96E)
3. **Форма длинная vs короткая:** 2 поля (телефон + имя) vs 4 поля (+ площадь + комментарий)
4. **Расположение калькулятора:** В хедере vs на лендинге vs отдельная страница
5. **Триггер exit-попапа:** Чек-лист vs «Оставьте телефон, мы перезвоним»

## 6. Чек-лист перед запуском

### SEO
- [ ] Уникальный title (≤60 символов) и description (≤160) на каждой странице
- [ ] H1 на каждой странице (ровно один)
- [ ] Alt-тексты у всех изображений
- [ ] `sitemap.xml` — все страницы
- [ ] `robots.txt` — закрыть служебные страницы
- [ ] Schema.org (LocalBusiness, Product, FAQPage, Article, BreadcrumbList)
- [ ] Open Graph / Twitter Card для всех страниц
- [ ] ЧПУ-ссылки (domkuhni.by/kuhnya-s-ostrovom, не /?p=123)
- [ ] `hreflang` если мультиязычный
- [ ] 301 редиректы со старых URL

### Скорость (Core Web Vitals)
- [ ] LCP < 2.5 секунд (оптимизировать hero-изображение: WebP, responsive, preload)
- [ ] FID < 100ms (делей критического JS, минификация)
- [ ] CLS < 0.1 (явные размеры изображений, шрифты с `font-display: swap`)
- [ ] WebP/AVIF для всех изображений
- [ ] Lazy-load для изображений ниже сгиба
- [ ] CDN (Cloudflare / Vercel Edge)
- [ ] Кэширование статики (1 год для хэшированных файлов)
- [ ] Preconnect к Google Fonts и аналитике
- [ ] Минификация HTML/CSS/JS
- [ ] CSS критические → инлайн, остальное defer

### Формы
- [ ] Honeypot (скрытое поле) — защита от ботов
- [ ] reCAPTCHA v3 (невидимая) — опционально
- [ ] Валидация на клиенте (пустые поля, формат телефона)
- [ ] Валидация на сервере (XSS, SQL injection)
- [ ] Сообщение об успехе (на странице, не alert)
- [ ] Интеграция с CRM (все поля, включая UTM)
- [ ] Автоответ на email / SMS после заявки
- [ ] 152-ФЗ / GDPR: галочка согласия на обработку
- [ ] Ограничение частоты отправки (rate limit)

### Мобильная версия
- [ ] Touch-friendly: все кнопки ≥48×48px
- [ ] Шрифты ≥16px (запретить zoom на iOS)
- [ ] Меню: гамбургер, тапы, без вложенности
- [ ] Формы: крупные поля, автофокус
- [ ] Изображения: responsive srcset
- [ ] Анимации: `prefers-reduced-motion`
- [ ] Тест на реальных устройствах (iPhone 12/14, Samsung S23, Pixel)

### Аналитика
- [ ] GA4 + GTM (или прямой код)
- [ ] Яндекс.Метрика (Вебвизор, Карта кликов, Цели)
- [ ] UTM-трекинг на все внешние ссылки
- [ ] События:
  - Scroll 25%/50%/75%/90%/100%
  - CTA click (каждая кнопка — отдельное событие)
  - Form submit (каждая форма)
  - Gallery/Portfolio interaction
  - Calculator step completion
  - Exit popup trigger (intent / time / scroll)
  - Conversion (заявка)
- [ ] Facebook Pixel (если есть реклама FB/Instagram)
- [ ] Yandex.Klenty / Calltouch (call tracking — опционально)

### Безопасность
- [ ] HTTPS (Let's Encrypt / Cloudflare)
- [ ] HSTS header
- [ ] Content-Security-Policy (CSP) header
- [ ] X-Frame-Options: DENY
- [ ] X-Content-Type-Options: nosniff
- [ ] Referrer-Policy: strict-origin-when-cross-origin
- [ ] CSRF-защита на формах
- [ ] Rate limiting на endpoint-ы форм
- [ ] Политика обработки данных (отдельная страница)
- [ ] Согласие на cookie

## 9. Рекомендации по визуалу

| Элемент | Формат | Размер | Оптимизация |
|---------|--------|--------|-------------|
| Hero-фон | WebP | ≤200KB | 1920×1080, 80% quality |
| Портфолио | WebP | ≤100KB | 600×450, lazy-load |
| Блог | WebP | ≤80KB | 600×338, lazy-load |
| Логотип | SVG/PNG | ≤20KB | Inline SVG предпочтительнее |
| Иконки | SVG | ≤2KB | Inline или спрайты |

**Где брать реальные фото:**
- Попросить клиентов поделиться фото готовых кухонь (взамен на скидку)
- Нанять предметного фотографа для съёмки 5–10 лучших проектов
- Видео-обзоры кухонь (YouTube Shorts / Reels) для соцсетей и встраивания на сайт
- **Никаких стоковых фото** — это убивает доверие в премиум-сегменте

## 10. Конверсионные механики (приоритет)

1. **Мини-калькулятор на главной** — 4 клика до контакта, мгновенная цена
2. **Exit-intent popup** — чек-лист «7 ошибок» в обмен на email
3. **Time-based popup** — через 45 секунд на сайте предложение рассчитать стоимость
4. **Scroll-trigger** — при 60% прокрутки sticky-баннер «Получить расчёт»
5. **WhatsApp/Telegram кнопки** — в хедере, в контактах, в футере
6. **Социальное доказательство** — реальные отзывы с фото, количество проектов, гарантия
7. **Прозрачность процесса** — таймлайн с фото каждого этапа
8. **Политика возврата / гарантия** — отдельным блоком у формы заявки
