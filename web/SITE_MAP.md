# Карта сайта — Дом кухни (Премиум)

## 1. Главная (Landing Page)
**URL:** `/`
**Цель:** Конверсия посетителя → лид (заявка на замер/расчёт)

| Блок | Контент | Примечание |
|------|---------|------------|
| **Header (sticky)** | Лого, навигация (6 ссылок), WhatsApp/Telegram, телефон, бургер | Прозрачный → белый при скролле |
| **Hero** | Фоновое изображение кухни (full-width), оверлей, бейдж «Собственное производство • Гомель», H1 с акцентным словом, подзаголовок, 2 CTA (синий/outline), доверительные триггеры (12 лет, 350+ проектов, гарантия 5 лет, рассрочка 0%) | CTA1: «Рассчитать стоимость» → #calculator; CTA2: «Смотреть портфолио» → #portfolio |
| **Trust Bar** | 4 счётчика (лет опыта, проектов, % клиентов, лет гарантии) с анимацией count-up | Числа подтягиваются при появлении |
| **Портфолио** | Фильтр по стилю (все/современный/классический/скандинавский/лофт), сетка карточек (6 шт), каждая: фото, overlay с площадью/бюджетом, название, описание, кнопка «Хочу такую же» | Использовать WebP/AVIF, lazy-load |
| **Процесс (таймлайн)** | 5 этапов: Замер → 3D-проект → Материалы → Производство → Установка. Каждый: номер, заголовок, описание, срок. Анимация появления при скролле | Вертикальная линия, активный этап подсвечен |
| **Материалы** | Табы: Фасады / Столешницы / Фурнитура. Сетка карточек с превью, названием, описанием, тегом уровня (эконом/стандарт/премиум/элит) | 3 таба, 4+3+3 карточки |
| **Мини-калькулятор** | 4 шага: тип кухни → фасад → площадь (range) → телефон. Шкала прогресса. После отправки — модал с ценой | Сбор контактов, UTM, отправка в CRM |
| **Отзывы** | Карусель отзывов (3+), звёзды, текст, автор + фото/аватар, тип кухни. Стрелки и точки-прогресс | Авто-пролистывание 6 сек |
| **Блог** | 3 карточки статей: тег (Гид/Тренды/Сравнение), заголовок, превью, ссылка «Читать →» | SEO-трафик, внутренняя перелинковка |
| **Финальный CTA** | Сплит-экран: слева текст + телефоны + мессенджеры + адрес; справа форма (имя, телефон, комментарий, политика, сабмит) | Honeypot-защита, отправка в CRM |
| **Footer** | 4 колонки: 1) лого + описание + соцсети (inst/yt/pinterest); 2) разделы; 3) услуги; 4) контакты. Нижняя полоса: копирайт + политика + соглашение + реквизиты | Schema LocalBusiness |
| **Cookie notice** | Плашка внизу: текст + кнопка «Принять» | localStorage, GDPR/152-ФЗ |
| **Exit popup** | «Уже уходите? Скачайте чек-лист 7 ошибок при заказе кухни» + форма email | Срабатывает: уход мыши, 45 сек, скролл 60% |

## 2. Портфолио / Проекты
**URL:** `/portfolio.html`
**Цель:** Демонстрация экспертизы, вдохновение, социальное доказательство

- **Фильтрация:** стиль × тип помещения (угловая, прямая, П-образная, остров) × бюджет (эконом/средний/премиум/люкс) × площадь (до 10/10-15/15-20/20+ м²)
- **Карточка проекта:** фото (до/после), площадь, тип, бюджет, материалы, сроки
- **Детальная страница проекта (`/project-{id}.html`):** галерея (карусель), полное описание, 3D-визуализация, отзыв клиента, кнопка «Получить подобный проект»
- **SEO:** Schema.org Article, хлебные крошки, rel=next/prev для пагинации

## 3. Как мы работаем
**URL:** `/process.html`
**Цель:** Снять тревожность, показать прозрачность, экспертизу и сроки

- **Интерактивный таймлайн** (5 этапов) с фото/видео процесса на каждом этапе
- **Гарантии и сертификаты:** блок с иконками (договор, гарантия 5 лет, страхование, сертификаты)
- **Команда:** фото реальных мастеров, дизайнеров, замерщиков (лица → доверие)
- **Калькулятор сроков:** выберите тип и площадь → получите срок изготовления
- **CTA** в конце каждого этапа

---

## 4. Материалы и фурнитура
**URL:** `/materials.html`
**Цель:** Обосновать ценообразование, продемонстрировать качество

- **Табы/секции:** Фасады, Столешницы, Фурнитура, Техника
- **Для каждого материала:** фото (крупно/макро), характеристики, износ, уход, срок службы, примеры в проектах (ссылка на портфолио)
- **Сравнительная таблица:** бюджет → стандарт → премиум → элит
- **Бренды:** Blum, Hettich, Grass, Caesarstone, Samsung, Bosch, Miele (с логотипами)
- **Фильтр «Подобрать материал»:** вопросы → рекомендация

---

## 5. Калькулятор / Конфигуратор
**URL:** `/calculator.html`
**Цель:** Лидогенерация + квалификация

- **Полная версия** (6 шагов):
  1. Тип кухни (4 варианта с иконками-схемами)
  2. Размеры (ширина × глубина, слайдеры)
  3. Материал фасада (8 вариантов с цветовыми превью)
  4. Столешница (3 варианта)
  5. Фурнитура (Blum / Hettich / Grass / бюджетная)
  6. Доп. опции (сушилка, карго, подсветка, барная стойка)
- **Визуализация:** меняется схема кухни в реальном времени
- **Цена:** динамический пересчёт при каждом изменении
- **Результат:** модал с ценой, разбивка, кнопка «Отправить заявку»
- **После отправки:** письмо с PDF-спецификацией + создание сделки в CRM

## 6. Отзывы и гарантии
**URL:** `/reviews.html`
**Цель:** Социальное доказательство, снятие возражений

- **Витрина отзывов:** сетка карточек + возможность оставить свой отзыв
- **Фильтр:** по стилю кухни, по типу, по оценке
- **Видео-отзывы:** Youtube-плеер (настоящие клиенты в своих кухнях)
- **Гарантии:** иконки + текстовые блоки (гарантия на мебель, на фурнитуру, на установку, страхование груза)
- **Кейсы «Было — Стало»:** слайдер до/после
- **FAQ:** Schema.org FAQPage — 10+ вопросов

---

## 7. Блог / Гиды (SEO)
**URL:** `/blog/`
**Цель:** Органический трафик, экспертное позиционирование

- **Рубрики:**
  - «Как выбрать кухню» — гид для начинающих
  - «Тренды дизайна» — ежегодные обновления
  - «Сравнение материалов» — ЛДСП vs МДФ vs Акрил vs Fenix vs Массив
  - «Уход и эксплуатация» — как сохранить кухню
  - «Эргономика» — правила планировки, рабочий треугольник
- **Формат:** заголовок H1, intro (200 символов), основная статья (1500-2500 слов), маркированные списки, таблицы, изображения, CTA-блок в конце (подписка / расчёт)
- **SEO:** Schema Article, OG, Twitter Card, внутренняя перелинковка, breadcrumbs, sitemap.xml
- **Email-подписка:** «Получите чек-лист при подписке»

---

## 8. Контакты
**URL:** `/contact.html`
**Цель:** Максимальное упрощение связи + сбор заявок

- **Контакты:** телефоны (ссылки tel:), email, часы работы, адрес с картой (Яндекс / Google)
- **Быстрые кнопки:** «Позвонить», WhatsApp, Telegram, Viber, Instagram
- **Форма заявки:** 3+ поля (имя, телефон, комментарий) + согласие + honeypot
- **Карта проезда:** интерактивная, маркер
- **Реквизиты:** ИНН, ОГРН, р/с (для юрлиц)

## Техническая структура

```
domkuhni.by/
├── index.html              # Главная (все блоки)
├── portfolio.html          # Портфолио
├── project-{id}.html       # Детальная проекта
├── process.html            # Как мы работаем
├── materials.html          # Материалы
├── calculator.html         # Полный конфигуратор
├── reviews.html            # Отзывы
├── blog/                   # Блог (рубрики)
├── contact.html            # Контакты
├── styles.css              # Дизайн-система
├── main.js                 # Все скрипты
├── slider.js               # Слайдер (legacy)
├── images/                 # Оптимизированные изображения
├── icons/                  # SVG-иконки
└── docs/                   # Документация
```

**Схема данных:** Schema.org LocalBusiness + Product + FAQPage + Article + BreadcrumbList
**Карта сайта:** `sitemap.xml` (авто-генерация)
**Robots:** `robots.txt`
